# KPSC OJ - Frontend PRD

## 1. 개요

Frontend는 사용자 인터페이스를 담당하며 KPSC 스타일 UI를 제공한다.

## 2. 기술 스택

* React
* Tailwind CSS
* TypeScript
* Zustand (상태 관리)

## 3. 주요 페이지

### 3.1 문제 목록

* 문제 리스트
* 난이도 표시
* 태그 필터

### 3.2 문제 상세

* 문제 설명
* 입력/출력
* 제한 조건

### 3.3 제출 페이지

* 코드 입력
* 언어 선택
* 제출 버튼

### 3.4 제출 결과

* 상태 표시
* 실행 시간 / 메모리

### 3.5 마이페이지

* 제출 기록
* 맞은 문제
* 티어

### 3.6 랭킹

* 유저 순위
* 점수 기반 정렬

## 4. UX 요구사항

* 빠른 반응 속도
* 직관적인 문제 풀이 흐름
* 다크 모드 지원

## 5. API 연동

* Backend REST API 사용

## 6. 상태 관리

* 인증 상태
* 문제 목록 캐싱

## 7. 코드 구조

* components/
* pages/ (app/)
* hooks/
* services/api

## 8. 확장 계획

* 코드 에디터 (Monaco)
* 실시간 제출 상태
